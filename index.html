Jessnet
Desktop Web App

As a System Administrator, I wanted an easy way to check on the status of our servers, so I created Jessnet (named after our network administrator, Jessica). This program pings all of our servers every 30 seconds and shows if any of them have timed out. I used Node.js along with Electron to create what looks like a desktop widget. Without node.js, the buttons will be unusuable in this example, but you can see the ping functionality.
